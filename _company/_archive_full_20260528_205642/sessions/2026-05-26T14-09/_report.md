🎯 **MrBeast 유튜브 전략**

- **용도**: 고조류량 유튜브 채널 성장 전략 설계  
- **연결 관계**: **1-hop**으로 **Dashboard Kit** (SaaS 관리 대시보드)와 **Landing Kit** (랜딩페이지)에 간접적 영향  

🎯 **AI 1인 기업: 자동화 챕터 1**  
- **주제**: 비전공자 대상 AI 비즈니스 교육 프레임워크  
- **용도**: 초보 개발자/창업자의 지능형 자동화 도입  
- **연결 관계**: **Dashboard Kit** (관리자 인터페이스 구현 예제 포함)와 **Portfolio Kit** (크리에이터 포트폴리오)와 개념적 연계  

🎯 **Dashboard Kit — SaaS·관리자 대시보드**  
- **주제**: Multi-tiered UI 구성 (사이드바 + 탑바 + 메인)  
- **용도**: SaaS 운영자용 대시보드 빠른 개발  
- **연결 관계**:  
  - 🔗 **Landing Kit**: 랜딩페이지와 대시보드의 UX 흐름 일관성 유지  
  - 🔗 **Portfolio Kit**: 크리에이터 포트폴리오의 네비게이션 구조 공유  
  - 🔗 **Mobile Kit**: 모바일 앱 Bottom Tab Navigator와 대시보드 탑바의 디자인 언어 연동  

🔗 **Landing Kit — 1인 기업·SaaS 랜딩 페이지**  
- **주제**: 검증된 6-섹션 구조 (로고 + 메뉴 + CTA)  
- **용도**: 5분 이내 개인 브랜딩 사이트 제작  
- **연결 관계**: Dashboard Kit의 UI 요소 (탭바, 사이드바)와 공통 디자인 시스템 적용  

🔗 **Portfolio Kit — 1인 크리에이터·프리랜서**  
- **주제**: Sticky 네비게이션 + 프로젝트 그리드 레이아웃  
- **용도**: 크리에이터/프리랜서 포트폴리오 템플릿  
- **연결 관계**: Dashboard Kit의 레이아웃 구성 요소 재사용  

🔗 **Mobile Kit (Expo) — 진짜 모바일 앱**  
- **주제**: Bottom Tab Navigator (3 화면) 구조  
- **용도**: React Native 기반 모바일 앱 템플릿  
- **연결 관계**: Dashboard Kit의 탑바 디자인 언어와 일관된 색상/타이포그래피 시스템 공유
